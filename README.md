# 🦉 ComOwl – LAN Messenger

**ComOwl** is a powerful LAN-based communication platform that enables users to chat, call, and play games seamlessly within the same network — no internet required.

---

## 🚀 Features

* 💬 **Real-time Messaging**
  Instantly send and receive messages between users connected to the same network.

* 📞 **Audio & Video Calling**
  Communicate through smooth voice and video calls.

* 🎮 **Dino Rush Game**
  Play the built-in fun game directly inside the messenger.

* 👥 **Group Chat**
  Create groups and communicate with multiple users at once.

* 🎤 **Voice Clip Sharing**
  Record and send voice messages quickly.

---

## 🧠 How It Works

ComOwl uses a **Local Area Network (LAN)** to connect devices.
All users must be connected to the same Wi-Fi or local network to communicate.

---

## 🛠️ Tech Stack

* Java
* JavaFX
* Socket Programming
* TCP/UDP Networking
